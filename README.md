## สรุปเนื้อหาแต่ละ part

### Part 1: Scene Setup
- ทดสอบการใช้งาน `Asset Store` และ `Package Manager` เพื่อช่วยในการทำเกม
- ลองใช้งาน `FPS Controller`
- สร้าง prototype ฉาก ด้วย `ProBuilder`
- ลองจัดแสงเงาเกมผีเบื้องต้นและวิธีการทำให้ฉากมืด
